### With Open Server Panel

- Go to `Open Server Panel`
- Settings
- Domains
- Write name: `app-24-5`
- Select folder `OSPanel\domains\app-24-5`
- Save
- Go to `app-24-5/`


### Use php

```bash
$ php -S localhost:8080
```
